# 📝 Console To-Do List Application

A simple, crash-resistant Python CLI application designed to manage daily tasks using basic CRUD operations (Create, Read, Delete).

## ✨ Key Features
* **Error Handling:** Implements `try-except` blocks to smoothly handle invalid user inputs and prevent interface crashes.
* **UX Optimization:** Dynamically displays remaining tasks within the application menus for better user guidance.
* **Clean Code:** Structured cleanly with reusable functions to ensure code maintainability.

## 🛠️ Tech Stack
* Python 3.x (Lists, Loops, Functions, and Exception Handling)
